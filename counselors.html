02Worked example

Getting a student to Capstone CS.

A 10th-grader wanting to end senior year in Capstone CS would typically take the path below. Computer Skills in 9th is recommended, not required. Order is negotiable — a student who already programs can skip Computer Skills and start at Intro to CS in 10th, or take Intro to CS in 11th instead.

Flexible order · no hard prerequisites above 9th grade

  1. 9th · recommended

    Computer Skills

    CHS / CVHS · Randy Macdonald. The soft on-ramp — strongly recommended, not a hard prereq.

  2. 10th

    Intro to CS

    CHS · Randy Macdonald. First full programming year; Python or JavaScript.

  3. 11th

    Advanced CS

    CHS · Randy Macdonald. Data structures, algorithms, collaborative git-based work.

  4. 12th · capstone

    Capstone CS

    CHS · Randy Macdonald. Year-long self-directed software project.

Arrows between boxes represent the typical path (9th → 10th → 11th → 12th). A student who already programs can skip Computer Skills and start at Intro to CS in 10th.

03Cross-building & scheduling

One course at the other building is normal.

A student may take one course per year at the other building — the program treats cross-site enrollment as a normal option, not an exception. Scheduling is a real constraint, though: bell times, transport, and period availability all have to line up. Talk to the registrar early so the student’s schedule survives the build.
